###Using Git & GitHub

**Global setup:**

- git config --global user.name "Your Name"
- git config --global user.email "your_email@address.com"


**Basic git commands:**
		 
* git init
* git add
* git commit
* git status
* git reset
* git log
* git clone
* git checkout
* git branch
* git push
* git pull
* git remote
